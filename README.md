A responsive, personal portfolio website designed and developed to showcase my skills, projects, and educational background as an aspiring AI & ML Developer.

## 🛠️ Technologies Used
- **HTML5**: For structuring the web content.
- **CSS3**: For styling, responsive design (Flexbox/Grid), and glassmorphism effects.
- **JavaScript (ES6+)**: For interactive elements like mobile navigation, sticky headers, scroll animations, and dynamic contact form handling.
- **FontAwesome**: For scalable vector icons.
- **Google Fonts**: Custom typography using 'Inter' and 'Outfit' fonts.

## 🌟 Features
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Modern UI Elements**: Clean aesthetics featuring glassmorphism (frosted glass) effects and smooth hover transitions.
- **Scroll Animations**: Dynamic elements that fade in as the user scrolls down the page using the Intersection Observer API.
- **Interactive Navigation**: Sticky navigation bar with active section highlighting and a mobile hamburger menu.
- **Contact Form**: Direct email functionality via standard mail client integration.

## 🔧 Setup and Installation
To run this project locally, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SayantikaBera-05/your-repo-name.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd Portfolio-Website
   ```

3. **Open the project:**
   Simply open the `index.html` file in your preferred web browser. Alternatively, you can use an extension like **Live Server** in VS Code for a better development experience.
